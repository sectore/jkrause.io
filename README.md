# jkrause.io

## Installation

### (a) `Nix` users

`cd` into root folder of the project.

```bash
nix develop
```

### (b) Non `Nix` users

Install [Node](https://nodejs.org) and [pnpm](https://pnpm.io/).

### Install dependencies

```bash
pnpm i 
```

## Local development

```bash
pnpm dev
```

Open http://localhost:5173/ in your browser.

## Local build

```bash
pnpm build
```
