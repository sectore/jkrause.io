# [jkrause.io](http://www.jkrause.io)
Just my personal portfolio and blog.

_Installation:_
1. Make sure [Git](http://git-scm.com/), [Ruby](http://www.ruby-lang.org/) and [Bundler](http://gembundler.com/) are installed on your machine.

2. Run in Terminal

  ```shell
  git clone https://github.com/sectore/jkrause.io
  cd jkrause.de
  bundle install
  jekyll serve
  ```

3. Open [http://localhost:9999](http://localhost:9999)
